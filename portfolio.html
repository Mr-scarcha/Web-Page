Adrian Huerta, Marco Antonio Huerta, Vladimir Gonzalez, Carlos Germán Solís, Olegario Porras y Juan Carlos Rodríguez.
Para poder llevar a cabo el proyecto se investigó el funcionamiento de un láser CNC, los componentes que se necesitaban, se hizo un diseño en el Software SolidWorks de cada una de las piezas para poder realizar los planos de fabricación, además se simuló su armado, se realizó un Diagrama de Proceso, se realizó la Boom List para el proyecto y se realizaron cotizaciones.

SIMULACIÓN DE PROCESO

La simulación fue realizada en el Software de nombre FlexSim y en este lo que podemos apreciar es la semejanza al Proceso de Manufactura de una empresa dedicada al giro de Elaboración de Pinturas. Para poder realizarlo se investigó el proceso con personal que ha laborado en el giro, proporcionarón tiempos estimados de producción, una distribución de planta, número de personal laborando en piso, etc.
Para esto se utilizarón algunos recursos como:

  • Source - Este recurso simuló la entrada de Materia Prima al Proceso de Manufactura
  • Processor - Cómo su nombre lo indica simuló algunos pasos del Proceso como calidad, los datos que se utilizarón fueron los tiempos que transcurria para realizar el proceso.
  • Combinator - De lo que se encargaba el recurso era combinar dos elementos del proceso, por ejemplo: poner en una tarima cierta cantidad de producto envasado y sellado.
  • Separator - Aquí se simuló el proceso de envasado
  • Queue - Este recurso era encargado de simular un almacén intermedio y un almacén final antes de la salida del producto terminado.
  • Operator - Simulaba a los Operadores Involucrados en el Proceso, a este recurso se le puede introducir datos como a que velocidad puede moverse, cuanto puede cargar, entre otras características.
  • Transporter - Simula un Montacargas y al igual que el recurso Operator se le puede añadir ciertas características
  • Sink - Se encarga de Terminar el Proceso como Producto Terminado

FUNDICIÓN

Para poder llevar acabo este Proyecto, lo primero que se hizo fue realizar una Investigación sobre el Proceso de Fundición, Medidas de Seguridad, Materiales, Herramientas, sin olvidar los Tipos de Moldes con los que se puede trabajar.
El material que se escogio para Fundir fue Aluminio, esto debido al fácil acceso del material, la cantidad de material a fundir, el horno de crisol a utilizar y principalmente el punto de fusión, que es de 660.3 °C.
La segunda decisión a tomar fue el tipo de Molde a utilizar que fue Molde de Arena Verde, debido al presupuesto con el que se contaba, la cantidad de material que se podía usar, etc.
Se denomina Molde de Arena Verde debido a contiene humedad durante el vaciado, los materiales requeridos para este tipo de moldeo son 90% Arena de Sílice 7% de Arcilla o Aglutinante (Bentonita) y 3% de Agua.
La tercera decisión que se tomó fue el modelo a fundir, para estó usamos un avión de juguete como Modelo.
Una vez tomadas estas decisione se pasó a:

  • Colocar el Aluminio en el Crisol y Calentar el Material.
  • Realizar el Mezclado de la Arena Sílica, el Aglutinante y Agua.
  • Hacer los Moldes, para esto el modelo se coloca en la caja de moldeo que se encuentra con el mecanismo de cierre abajo, se le agrega polvo separador, se rellena la caja con arena ya preparada y refinada, una vez esto, se comienza con apizonar la Arena(se refiere compactar la arena hasta llegar a cierta dureza) y se voltea la caja con el modelo para arriba.
    Después se cubre con la segunda caja, antes de rellenar la caja con arena se colocan los bebederos que serán los canales por donde entra el material, se continua con el proceso que se hizo en la primera caja, terminado esto se retiran los tubos para hacer los bebederos también se retira el modelo y se crean los canales que llevarán el material que entra del bebedero al modelo en la arena y el bebedero de salida.
  • Antes de realizar el vaciado de Material en el molde, se "Cucharea" el material fundido para sacar la escoria.
  • Se realiza el vaciado del material.
  • Se deja que se enfríe un poco para después destruir el molde.
  • Se comienza con el mecanizado de la pieza.

FABRICACIÓN DE LOCKER

El Proceso de Conformado en Frío se refiere al trabajo realizado a una temperatura ambiente o debajo de la termperatura para la recristalización.
Cómo proyecto del proceso de Conformado en Frío, manejo de Soldadura, Herramientas y Maquinaría, se opto por realizar la fabricación de un locker dónde se utlizó:

  • Lámina Calibre 22
  • Dobladora
  • Máquina de Soldar MIG
  • Remachadora
  • Taladro
  • Pulidora
  • Discos de Corte y para Pulir
  • Brocas de 3/16 y 5/32
  • Lentes
  • Guantes y más.

El Proceso de Fabricación del Locker fue:

  • Realizar los dobleces de la Lamina que conformarían las paredes laterales y trasera

  • Se realizarón las tapas Superior e Inferior, una vez que coincidian con el cuerpo del locker pasó al Área de Soldadura
  • Se soldarón ambas tapas en sus respectivos lugares.
  • Se fabricarón las puertas de cada uno de los casilleros y se marcarón dónde es que quedarían los orificios para el flujo de aire y chapa
  • Se dió paso a la perforación de cada una de las puertas y se limarón asperezas así como agrandar el orificio para la chapa
  • se marcarón los orificios para las charolas de cada división del locker.
  • Se marcarón y perforarón los agujeros para colocar las bisagras.
  • Pasó el locker al Área de Soldadura para colocar los angulos que funcionarían como patas.
  • Se fue colocando y remachando cada una de las partes del locker; Charolas, Puertas, Bisagras y Cerradura
  • Se pulierón los residuos de la soldadura, además lijar el locker para quitar el zonas con oxidación
  • Luego pasó al Área de Pintura, donde se aplico la primera capa de Primer por parte interna como externa
  • Seguido de eso, se pasó a Pintar el locker del Color Rojo por parte interna y externa
  • Como último paso y seco el locker, se remacharón las manijas de cada una de las puertas.
